Minhas Skills

Descubra as habilidades e ferramentas que domino e que me permitem criar soluções criativas e funcionais para meus clientes.

  • Stacks mais utilizadas

    Minhas principais stacks de desenvolvimento web: HTML5, CSS3 e JavaScript formam a base para criar interfaces interativas. Bootstrap agiliza o design responsivo. Git e GitHub garantem versionamento e colaboração eficiente.

  • Stacks conhecidas

    Possuo conhecimentos básicos em PHP, ReactJS, Tailwind e MariaDB para desenvolvimento web, mas ainda não domino essas stacks. Estou em processo de aprendizado para aprimorar minhas habilidades nesses ambientes tecnológicos.

  • Web Design

    As ferramentas de web design que mais utilizo são o Figma para design de interfaces e prototipagem, o Canva para criação de gráficos e mídias sociais, e o GIMP para edição e manipulação de imagens. Uma combinação poderosa para o desenvolvimento web criativo.

Projetos em Destaque

De ideias abstratas a projetos bem-sucedidos, aqui estão alguns dos meus trabalhos mais recentes

  • Coletivo Atômico

    Portfolio para o estúdio de ilustração Coletivo Atômico, focado em exibir seus emocionantes projetos, tanto em parceria com outros estúdios de animação, quanto os pessoais. O site foi cuidadosamente construído com as tecnologias HTML5, CSS3 e Bootstrap, garantindo uma experiência visualmente impactante e uma navegação suave em todas as telas e dispositivos.

    Conferir →

  • Liga SmashAju

    Aplicação web do grupo de e-sports Smashaju, desenvolvida do zero com HTML5, CSS3, Bootstrap, JavaScript e Node.js. Funcionalidades: sistema de pontos para adquirir produtos na área de shop, visualização de posição no ranking estadual e área de informações do grupo.

  • Processo Seletivo de Nossa Senhora do Socorro

    aplicação web desenvolvida para um processo seletivo, sob demanda da Secretaria de Educação de Nossa Senhora do Socorro. O front-end foi criado utilizando as tecnologias HTML5, CSS3, Bootstrap e JavaScript, enquanto o back-end foi implementado em PHP, com o banco de dados em MariaDB. Essa aplicação busca atender às necessidades específicas do processo seletivo, fornecendo uma interface responsiva e funcional, juntamente com uma estrutura robusta para gerenciar os dados dos candidatos e as etapas do processo seletivo.
