Projetos em Destaque

De ideias abstratas a projetos bem-sucedidos, aqui estão alguns dos meus trabalhos mais recentes

  • Portfolio para uma empresa de ilustração

    Sistema web desenvolvido para a empresa Coletivo Atômico, focado em exibir seus emocionantes projetos, tanto em parceria com outros estúdios de animação, quanto os pessoais. O site foi cuidadosamente construído com as tecnologias HTML5, CSS3 e Bootstrap, garantindo uma experiência visualmente impactante e uma navegação suave em todas as telas e dispositivos.

  • Desenvolvimento de sistema web para grupo de e-sports

    Aplicação web do grupo de e-sports Smashaju, desenvolvida do zero com HTML5, CSS3, Bootstrap, JavaScript e Node.js. Funcionalidades: sistema de pontos para adquirir produtos na área de shop, visualização de posição no ranking estadual e área de informações do grupo.

    Leia mais →

  • Desenvolvimento de um Sistema para Processo Seletivo

    aplicação web desenvolvida para um processo seletivo, sob demanda da Secretaria de Educação de Nossa Senhora do Socorro. O front-end foi criado utilizando as tecnologias HTML5, CSS3, Bootstrap e JavaScript, enquanto o back-end foi implementado em PHP, com o banco de dados em MariaDB. Essa aplicação busca atender às necessidades específicas do processo seletivo, fornecendo uma interface responsiva e funcional, juntamente com uma estrutura robusta para gerenciar os dados dos candidatos e as etapas do processo seletivo.
