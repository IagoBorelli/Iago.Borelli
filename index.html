Minhas Skills

Descubra as habilidades e ferramentas que domino e que me permitem criar soluções criativas e funcionais para meus clientes.

  • Front-end Stacks

    Minhas principais stacks de desenvolvimento Front-end: HTML5, CSS3 e JavaScript, Typescript, React e Next.js formam a base para criar interfaces interativas. Bootstrap e Tailwind agiliza o design responsivo. A ferramenta de web design que mais utilizo é o Figma para design de interfaces e prototipagem.

  • Back-end Stacks

    Minhas principais stacks de desenvolvimento Back-end: PHP e Laravel oferecem uma base sólida para desenvolvimento rápido e escalável. Com PostgreSQL, MariaDB e MySQL, garanto a segurança e integridade dos dados, enquanto o Apache assegura a entrega confiável das minhas aplicações web.

  • Stacks Adicionais

    Minhas stacks adicionais de ferramentas incluem: Git, GitHub e GitLab, essenciais para controle de versão e colaboração eficiente em projetos. Insomnia, um cliente de API, facilita testes e debugging de APIs REST. Obsidian é minha escolha para organização de documentações, aprimorando minha produtividade com uma interface intuitiva e flexível.

Projetos em Destaque

De ideias abstratas a projetos bem-sucedidos, aqui estão alguns dos meus trabalhos mais recentes

  • Processo Seletivo de Nossa Senhora do Socorro

    aplicação web desenvolvida para um processo seletivo, sob demanda da Secretaria de Educação de Nossa Senhora do Socorro. O front-end foi criado utilizando as tecnologias HTML5, CSS3, Bootstrap e JavaScript, enquanto o back-end foi implementado em PHP, com o banco de dados em MariaDB. Essa aplicação busca atender às necessidades específicas do processo seletivo, fornecendo uma interface responsiva e funcional, juntamente com uma estrutura robusta para gerenciar os dados dos candidatos e as etapas do processo seletivo.

  • In.Orbit

    O in.orbit foi desenvolvido durante a Next Level Week (NLW) da Rocketseat. Este projeto visa criar um gerenciador de metas pessoais. Utilizei React no front-end e Nodejs no back-end e outras ferramentas do ecossistema JavaScript.

  • Turminha de Siri

    Landing Page educativa do projeto Turminha do Siri, sob demanda da Secretaria de Educação de Nossa Senhora do Socorro. O front-end foi criado utilizando as tecnologias HTML5, CSS3, Bootstrap e JavaScript. O portal é direcionado a toda a comunidade escolar e conta com jogos educativos online, materiais didáticos para download, acesso aos episódios da série.

  • SIGEF

    Sistema de gerenciamento de folgas, sob demanda da Secretária de Nossa Senhora do Socorro. Para o front-end utilizei as tecnologias HTML5, CSS3, Bootstrap e Javascript. Esse sistema busca automatizar as solicitações de folgas dos funcionários, fornecendo uma inteface funcional e responsiva.

  • Dashboard de atividades

    A aplicação web foi desenvolvida sob demanda da Fundação Municipal de Formação para o Trabalho (FUNDAT) e tem como objetivo gerar relatórios detalhados sobre as atividades realizadas pela instituição. O front-end utiliza as tecnologias HTML5, CSS3, Bootstrap e JavaScript, enquanto o back-end foi implementado em PHP, com o banco de dados MariaDB. A aplicação foi projetada para atender às necessidades específicas da FUNDAT, oferecendo uma interface responsiva e intuitiva, além de uma estrutura robusta para o gerenciamento eficiente dos dados das atividades.
